O YouTube Data Cloud é uma ferramenta para coleta e análise de dados do YouTube.

É possível fazer uma coleta de dados a partir de um vídeo ou de um termo de busca. Os dados coletados podem ser visualizados em forma de lista ou de grafo. Também é possível fazer o download das tabelas.

O modo de coleta salva todos os dados de todos os vídes recomendados, enquando o de navegação coleta apenas o dos vídeos selecionados

Para utilizar é necesário ter uma chave da YouTube Data API (v3). É possível acessar a documentação da API aqui. Para criar uma chave da API siga o tutorial aqui
